# Infrastructure as Code (IaC) - Secure Web Access Transformation

## Project Overview
[cite_start]This repository demonstrates a foundational approach to building secure, reliable, and scalable infrastructure using **Terraform**[cite: 26, 34]. [cite_start]It aligns with the goal of creating "codified guardrails" to protect services from security threats[cite: 24, 37].

## Key Features & Skills Demonstrated
* [cite_start]**Infrastructure as Code (IaC):** Defined a web server environment using modular Terraform scripts, ensuring repeatability[cite: 34, 52].
* [cite_start]**Security-First Mindset:** Implemented **Security Groups** as codified guardrails to enforce secure web traffic (HTTPS) and restricted SSH access, following secure coding principles.
* [cite_start]**Automation & Scripting:** Developed a **Bash deployment script** to automate task validation, showcasing a commitment to DevOps practices[cite: 46, 53, 57].
* [cite_start]**Operational Excellence:** Integrated **Outputs** for real-time tracking of infrastructure metadata to assist in live service diagnostics and troubleshooting[cite: 36, 45].

## Technologies Used
* [cite_start]**Terraform** (IaC Framework) [cite: 34, 52]
* **AWS** (Cloud Platform)
* [cite_start]**Git** (Version Control & Collaboration) 
* [cite_start]**Agile Principles:** Managed tasks with a focus on problem-solving and collaboration[cite: 55, 56, 59].

## How to Run
1. Ensure Terraform is installed.
2. [cite_start]Run `./deploy.sh` to initialize and validate the configuration[cite: 53].
3. Execute `terraform apply` to deploy the secure infrastructure.
