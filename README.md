# Infrastructure as Code (IaC) - Secure Web Access Transformation

## Project Overview
This repository demonstrates a foundational approach to building secure, reliable, and scalable infrastructure using **Terraform**. It aligns with the goal of creating "codified guardrails" to protect services from security threats.

## Key Features & Skills Demonstrated
* **Infrastructure as Code (IaC):** Defined a web server environment using modular Terraform scripts, ensuring repeatability.
* **Security-First Mindset:** Implemented **Security Groups** as codified guardrails to enforce secure web traffic (HTTPS) and restricted SSH access, following secure coding principles.
* **Automation & Scripting:** Developed a **Bash deployment script** to automate task validation, showcasing a commitment to DevOps practices.
* **Operational Excellence:** Integrated **Outputs** for real-time tracking of infrastructure metadata to assist in live service diagnostics and troubleshooting.

## Technologies Used
* **Terraform** (IaC Framework)
* **AWS** (Cloud Platform)
* **Git** (Version Control & Collaboration)
* **Agile Principles:** Managed tasks with a focus on problem-solving and collaboration.

## How to Run
1. Ensure Terraform is installed.
2. Run `./deploy.sh` to initialize and validate the configuration.
3. Execute `terraform apply` to deploy the secure infrastructure.
